# Global Banka Botu\n\nTürkçe ekonomi & banka botu. Prefix: `x!`\n\nKurulum:\n1. Node.js >= 20 yükleyin.\n2. Bu dosyaları repoya ekleyin.\n3. `.env.example`'i `.env` olarak kopyalayın ve BOT_TOKEN girin.\n4. (İsteğe bağlı) MongoDB kullanmak isterseniz MONGODB_URI girin.\n5. `npm install`\n6. `npm start`\n\nNotlar:\n- Admin ID: 1450895145271558245 (sadece bu kullanıcı admin komutlarını kullanabilir).\n- Loglar LOG_CHANNEL_ID varsa o kanala gönderilir.\n- Veri depolama: MongoDB varsa Mongo, yoksa `db.json` (repo kökünde) kullanılır.
